==================
General Information
==================

ID10-BEATS is SESAME's beamline for X-ray tomography. Is is optimized for absorption and phase contrast full-field radiography and tomography with hard X-rays.

Apply for beamtime
------------------

Visit the `SESAME User Guide <https://www.sesame.org.jo/for-users/user-guide>`_ for information regarding proposal submission and calls for proposals.
